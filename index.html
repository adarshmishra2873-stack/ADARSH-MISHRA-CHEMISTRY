ALKANES (CₙH₂ₙ₊₂)
Saturated Hydrocarbons — Free Radical Mechanism (FRM)
🏗️ PREPARATION OF ALKANES
1 Alkenes/Alkynes → H₂ / Ni or Pt or Pd Alkane ✅
2 R-X (Alkyl Halide) → Zn/H⁺ or dil HCl / Zn R-H (Alkane)
3 2R-X (Wurtz Rxn) → Na / Dry Ether R-R + 2NaX
4 R-COONa (Decarboxylation) → NaOH + CaO / Δ (Soda Lime) R-H
5 R-COONa (Kolbe's Electrolysis) → Electrolysis R-R + CO₂ (Anode)
6 C=O (Clemmensen) → Zn-Hg / Conc HCl CH₂ (Alkane)
7 C=O (Wolf-Kishner) → NH₂-NH₂ / KOH / Δ CH₂
🧠 Wurtz Trick याद करो:
Wurtz = Wही carbon दोगुना करता है! (R-X → R-R, carbon double ho jaata hai)
3° R-X mainly disproportion deta hai → better result nahi
ROR: I > Br > Cl ≠ F (F nahi deta Wurtz)
⚗️ CHEMICAL PROPERTIES
1 R-H + X₂ → hν (Free Radical) R-X + HX (Halogenation)
2 R-H + I₂ ⇌ HNO₃ (oxidizer) R-I + HI (Iodination)
3 CH₄ + O₂ → Cu / 523K / 100 atm 2CH₃OH (Controlled Oxidation)
4 CH₄ + O₂ → Mo₂O₃ / Δ HCHO + H₂O
5 CH₄ + H₂O → Ni / Δ CO + 3H₂ (Water Gas)
6 C₆H₁₄ (Pyrolysis) → 773K C₆H₁₂ + H₂ / C₄H₈ + C₂H₆ etc.
⚠️ Halogenation Reactivity याद करो:
F₂ > Cl₂ > Br₂ > I₂ (Reactivity — F most reactive)
F₂ > Cl₂ > Br₂ > I₂ (Selectivity उल्टा — Br most selective)
Trick: "Fast kaam karne wale ghalati karte hain" → F fast = less selective

ALKENES (CₙH₂ₙ)
Electrophilic Addition — Markovnikov's Rule
🏗️ PREPARATION OF ALKENES
1 Alcohol (Dehydration) → Conc H₂SO₄ / H⁺ / Δ Alkene (Saytzeff Major)
2 R-X (Dehydrohalogenation) → alc. KOH Alkene (Saytzeff)
3 Vicinal Dihalide → Zn / Dehalogenation Alkene
4 Alkyne (Partial Hydrogenation) → H₂ / Pd-BaSO₄ / Quinoline (Lindlar) cis-Alkene
5 Alkyne (Birch) → Na + Liq. NH₃ trans-Alkene
🧠 cis vs trans Trick:
Lindlar = L = Low (liquid reagent nahi) = cis
Na/NH₃ = Birch = trans (Anti addition)
याद: "Lindlar cis deta, Birch trans deta"
⚗️ CHEMICAL PROPERTIES (Addition Reactions)
1 Alkene + HX → Markovnikov R-CH-CH₃ (X on more substituted C)
2 Alkene + HBr → Peroxide (H₂O₂ or PhCOOCOPh) Anti-Markovnikov (Br on less sub. C)
3 Alkene + H₂SO₄ → Cold Conc H₂SO₄ Alkyl Hydrogen Sulphate
4 Alkene + H₂O → dil H₂SO₄ / H₃O⁺ Alcohol (Markovnikov)
5 Alkene + X₂ → CCl₄ (non-polar) Dihalide (Anti Addition)
6 Alkene + Br₂ → CCl₄ Reddish orange → Colourless (Test for unsaturation)
7 Alkene + KMnO₄ → Dil. Alk. KMnO₄ (Baeyer's) Vicinal Diol (syn addition)
8 Alkene + O₃ (Ozonolysis) → 1) O₃+CCl₄/Zn 2) +H₂O 3) +Me₂S Two carbonyls (C=O + O=C)
⚠️ Peroxide Rule याद रखो:
HCl और HI पर peroxide काम नहीं करता!
Only HBr + Peroxide → Anti-Markovnikov
Trick: "HBr hi karta hai, HCl-HI nahi"

ALKYNES (CₙH₂ₙ₋₂)
Terminal Alkynes Acidic — sp Carbon
🏗️ PREPARATION
1 CaC₂ + H₂O → — Ca(OH)₂ + CH≡CH (Acetylene)
2 Mg₂C₃ + H₂O → — Mg(OH)₂ + CH₃-C≡CH (Propyne)
3 Vicinal Dihalide → ① alc. KOH ② NaNH₂ Alkyne
⚗️ CHEMICAL PROPERTIES
1 Alkyne + H₂ → Pd/BaSO₄ (Lindlar) cis-Alkene → Alkane
2 R-C≡CH + 2HX → Markovnikov (2 steps) Gem Dihalide (R-C(X₂)-CH₃)
3 R-C≡CH + H₂O (Kucherov) → dil H₂SO₄ / HgSO₄ Ketone (Methyl Ketone from terminal)
4 R-C≡C-H (Acidic) → Na R-C≡C-Na⁻ + H₂
5 R-C≡C-H (Tollens test) → AgNO₃ + NH₄OH R-C≡C-Ag (Silvery white ppt)
6 R-C≡C-H (Cuprous Cl test) → Cu₂Cl₂ / NH₄OH R-C≡C-Cu (Red ppt)
7 2CH≡CH (Coupling) → Cu₂Cl₂ / NH₄Cl CH≡C-CH=CH₂ (Vinyl acetylene)
8 3CH≡CH (Cyclic coupling) → RHT (Red Hot Tube) Benzene (🔥 important!)
🧠 Acidic Nature याद करो:
Alkyne (sp) > Alkene (sp²) > Alkane (sp³)
NaHCO₃ के साथ react नहीं करता (पर NaOH, NaH, Na से करता है)
Terminal alkyne = acidic, Internal = not acidic

AROMATIC HYDROCARBONS (Benzene)
Electrophilic Substitution Reaction (ESR) — EAS Mechanism
🏗️ PREPARATION
1 3CH≡CH → RHT Benzene (Coupling)
2 Ph-COOH (Decarboxylation) → NaOH + CaO Benzene
3 Ph-OH (Phenol) → Zn dust / Δ Benzene
⚗️ CHEMICAL PROPERTIES (ESR)
1 Benzene + X₂ (Halogenation) → X₂ + Fe or FeCl₃ / FeBr₃ Halobenzene + HX
2 Benzene (Nitration) → Conc HNO₃ + Conc H₂SO₄ (Mixed acid) Nitrobenzene
3 Benzene (Sulphonation) → Fuming H₂SO₄ / Oleum Benzenesulphonic acid
4 Benzene + R-X (FC Alkylation) → Anhy AlCl₃ Alkylbenzene + Rearrangement
5 Benzene + R-CO-Cl (FC Acylation) → Anhy AlCl₃ Ketone (NO rearrangement)
6 Benzene + H₂ (excess) → Excess H₂ / Ni / HP / HT Cyclohexane
7 Benzene + Cl₂ (excess) → Cl₂ + hν (excess) C₆H₆Cl₆ = BHC / Gammaxene / Lindane
8 Benzene + O₃ → O₃ + CCl₄ / Zn CHO-CHO (Glyoxal)
🧠 FC Alkylation vs Acylation:
Alkylation → Rearrangement होता है (carbocation) + poly-alkylation
Acylation → No rearrangement, No poly-acylation ✅
Trick: "A(cyl) = Accurate = Rearrangement nahi"
📍 DIRECTING EFFECTS
TypePositionExamples
+M (Activating) ortho + para -OH, -NH₂, -OR, -NHR, -NHCOR, -R
-M (Deactivating) meta -NO₂, -CN, -CHO, -COOH, -SO₃H, -COR
Halogens (Special) ortho + para -F, -Cl, -Br, -I (deactivating but o/p directors)
🧠 याद करने की Trick:
"जो +M है वो o/p देता है, जो -M है वो meta देता है"
Halogens Exception: Deactivating but ortho-para (resonance wins over induction)
Halogens = -I (deactivate) + +M (o/p) → Net = o/p director

HALOALKANES & HALOARENES
SN1 / SN2 / E1 / E2 — Nucleophilic Substitution
🏗️ PREPARATION
1 R-OH + HX → — R-X + H₂O (Rearrangement of C⁺)
2 R-OH + SOCl₂ → Darzen Rxn R-Cl + SO₂↑ + HCl↑ (Retention)
3 R-OH + PX₃ → Red P + X₂ R-X + H₃PO₃ (Inversion)
4 R-X + NaI (Finkelstein) → Acetone / DMSO R-I + NaX (X=Cl,Br)
5 R-X + AgF (Swartz) → AgF R-F + AgX
⚡ SN1 vs SN2 — Super Important!
SN1 SN2
Carbocation बनता है No Carbocation
RDS = C⁺ formation Only Transition State
Rearrangement ✅ No Rearrangement
Rate = k[RX]¹ (Order 1) Rate = k[RX][Nu] (Order 2)
ROR: 3°>2°>1° ROR: 1°>2°>3° (Less steric)
Racemic Mixture (Both retention + inversion) Walden Inversion (100%)
Polar Protic (H₂O, ROH, RCOOH) Polar Aprotic (DMSO, DMF, Acetone)
Weak nucleophile Strong nucleophile
🧠 SN1 vs SN2 Super Trick:
SN1 = "1 ka matlab SLOW" — 3° favors (stable C⁺)
SN2 = "2 ka matlab STRONG Nu" — 1° favors (less steric)
CH₂=CH-X (Vinyl) और Ph-X → NSR नहीं देते!

ALCOHOLS
1°, 2°, 3° — Lucas Test, Oxidation, Dehydration
🏗️ PREPARATION
1 Alkene + H₂O → dil H₂SO₄ / H₃O⁺ (Acid catalyzed) Alcohol (Markovnikov)
2 Alkene + BH₃ (HBC) → B₂H₆/THF → H₂O₂/OH⁻ Anti-Markovnikov Alcohol (syn)
3 Alkene + OMDM → Hg(OAc)₂/H₂O → NaBH₄ Markovnikov Alcohol (anti)
4 Carbonyl + H₂ → LiAlH₄ (LAH) or NaBH₄ Alcohol (Reduction)
⚗️ CHEMICAL PROPERTIES
1 R-OH + Na/Δ → — RO⁻Na⁺ + ½H₂ (Acidic nature)
2 1° R-OH + MOA → PCC/PDC/CrO₃+Py Aldehyde (RCHO)
3 2° R-OH + MOA → PCC/PDC/CrO₃+Py Ketone
4 1° R-OH + SOA → KMnO₄ / K₂Cr₂O₇ / Jones Carboxylic Acid
5 3° R-OH + SOA → — No oxidation (X)
6 R-OH + R'OH (Esterification) → H⁺ / Conc H₂SO₄ Ester + H₂O
7 1° R-OH (Dehydration) → Conc H₂SO₄ / 413K Ether (Intermolecular)
8 Any Alcohol (Dehydration) → Conc H₂SO₄ / 443K Alkene (Saytzeff)
🧪 LUCAS TEST (1°, 2°, 3° distinguish):
Reagent: Conc HCl + Anhy ZnCl₂
1° alcohol → No turbidity (takes long time)
2° alcohol → Turbidity in minutes
3° alcohol → Turbidity in seconds ✅
Trick: "3° = 3 second mein turbid!"
🧠 MOA vs SOA Oxidizing Agents:
MOA (Mild Oxidizing Agent) — stops at aldehyde: PCC, PDC, CrO₃+Pyridine, MnO₂ (allylic/benzylic), Cu/Ag at 300°C
SOA (Strong Oxidizing Agent) — goes to acid: KMnO₄, K₂Cr₂O₇, Na₂Cr₂O₇, CrO₃/H₂SO₄ (Jones), Conc HNO₃

ETHERS (R-O-R')
Williamson Synthesis — Electrophilic Substitution
🏗️ PREPARATION
1 1° R-OH (Dehydration) → Conc H₂SO₄ / 413K R-O-R (Ether)
2 1° R-X + R'-ONa (Williamson) → SN2 R-O-R' + NaX
3 2R-X + Ag₂O (dry) → — R-O-R (Ether)
⚠️ Williamson Synthesis Exception:
2° और 3° R-X → Alkene (Elimination) देता है, Ether नहीं!
Ph-X, CH₂=CH-X भी Williamson नहीं देते
⚗️ CHEMICAL PROPERTIES
1 R-O-R' + HX (excess) → Excess HX R-I + R'-I (Two halides)
2 R-O-R' + HX (HI > HBr > HCl) → More stable C⁺ R-X + R'-OH
3 Anisole + Cl₂ (ESR) → Br₂ / CH₃COOH or CS₂ Ortho + Para product (OCH₃ = o/p director)
4 Anisole + PCl₅ → — R-Cl + R'-Cl + POCl₃

7 R'-C≡N + RMgCl (Ketone) → 1) RMgCl 2) H₃O⁺ Ketone (R'COR)
8 R'-COCl + R₂Cd (Ketone) → Cadmium reagent Ketone

Aldehyde > Ketone (reactivity in NAR)

CARBOXYLIC ACIDS
HVZ Reaction, Hell-Volhard-Zelinsky — α-halogenation
🏗️ PREPARATION
1 1° Alcohol / Aldehyde → KMnO₄/H⁺ / CrO₃/H₂SO₄ / Conc HNO₃ Carboxylic Acid
2 Alkylbenzene → alk. KMnO₄ / Δ → H₃O⁺ Benzoic acid
3 R-CN (Hydrolysis) → H₃O⁺ (complete) R-COOH (Carboxylic acid)
4 R-CN (Partial Hydrolysis) → H₃O⁺ (partial) R-CONH₂ (Amide)
5 RMgX + CO₂ → 1) CO₂ 2) H₃O⁺ R-COOH
⚗️ CHEMICAL PROPERTIES
1 RCOOH + R'OH (Esterification) → H⁺ RCOOR' + H₂O
2 RCOOH + PCl₃/SOCl₂ → — R-CO-Cl (Acyl Chloride)
3 2 RCOOH (Anhydride) → P₂O₅ / Conc H₂SO₄ R-CO-O-CO-R (Anhydride)
4 RCOOH + NH₃ → Δ R-CO-NH₂ (Amide)
5 RCOOH + LAH → 1) LAH 2) H₂O R-CH₂OH (1° Alcohol)
6 RCOOH (Decarboxylation) → Soda lime / NaOH+CaO R-H (Alkane)
7 R-CH₂-COOH + Red P + X₂ (HVZ) → ① Red P + X₂ ② H₂O R-CHX-COOH (α-halogenation)
📊 Acidic Strength Order:
Acidic strength ∝ -M, -I effect and inversely ∝ +M, +I
F > Cl > Br > I (halogenated acids — F most acidic)
HCOOH > CH₃COOH (formic acid > acetic acid)
Benzoic acid < chlorobenzoic acid (o/p) < meta
Trick: Electron withdrawing groups = more acidic

AMINES
Basicity — Diazonium Salt — Carbylamine Test
🏗️ PREPARATION
1 R-NO₂ (Reduction) → Sn+HCl / Fe+HCl / H₂+Ni R-NH₂ (Primary Amine)
2 R-CN (Reduction) → H₂+Ni / Na(Hg)+ROH / LiAlH₄ R-CH₂-NH₂
3 NH₃ + R-X (Ammonolysis) → SN2 R-NH₂ → R₂NH → R₃N → R₄N⁺X⁻
4 R-CONH₂ (Hoffman degradation) → Br₂ / 4NaOH R-NH₂ (1 carbon कम!)
5 Gabriel Phthalimide → ① NaOH/KOH ② R-X ③ NaOH(aq) 1° Aliphatic Amine only
🧠 Hoffman Degradation Trick:
RCONH₂ → RNH₂ (carbon 1 kum ho jaata hai!)
Br₂ + 4NaOH (4NaOH = important!)
Stereochemistry remains same (retention)
⚗️ CHEMICAL PROPERTIES + BASICITY
1 R-NH₂ + CHCl₃/KOH (Carbylamine) → — R-NC (Isocyanide, foul smell) — 1° amine test
2 R-NH₂ + PhSO₂Cl (Hinsberg) → — Sulfonamide (Soluble in alkali) = 1° amine
3 R₂NH + PhSO₂Cl (Hinsberg) → — Insoluble sulfonamide = 2° amine
4 R₃N + PhSO₂Cl → — No Reaction = 3° amine
📊 BASICITY ORDER याद करो:
Gas Phase: 3° > 2° > 1° > NH₃
Aqueous Phase (R=Me): 2° > 3° > 1° > NH₃
Aqueous Phase (R=Et): 2° > 1° > 3° > NH₃
Aromatic Amines: Aliphatic >> Aromatic
SIP: Basicity decreases — 1° aromatic: m > p > o
SIR: Basicity increases — 2°,3° aromatic: o > m > p
🧠 Basicity Trick:
Aromatic amine < Aliphatic amine (lone pair delocalized in ring)
Aniline does NOT give FCR (Friedel Craft) — forms complex with AlCl₃
